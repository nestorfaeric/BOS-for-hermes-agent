# Skill: Traffic

Mettre en place **un** canal d'acquisition qui génère des leads de façon répétable. Déclenché quand `diagnosis` identifie un problème de trafic en phase PMF — pas assez de monde au sommet du funnel, ou pas de canal actif.

La plupart des entrepreneurs « font un peu de tout » et ne collectent jamais assez de données sur un levier. Ce skill impose la discipline inverse : **un canal, 90 jours**, avec BOS qui produit la majorité des livrables (copy, structure, analyse).

## Objectif

À la fin de la session : **(1)** un canal unique choisi et justifié, **(2)** le canal initialisé selon les best practices, **(3)** au moins un premier livrable concret produit dans cette conversation (post, script, structure de campagne, page, etc.), **(4)** fichiers Core et Output mis à jour.

**Critères de succès :** recommandation claire + setup actionnable + première pièce publiable + traçabilité dans `Output/` et `Core/`.

## Croyances

- **1 seul canal pendant 90 jours.** Si l'entrepreneur en ouvre plusieurs, BOS interdit les autres et recentre — sinon zéro apprentissage, zéro comparaison honnête.
- **L'IA fait la majorité du travail** (copy, structure, analyse). L'humain fournit le contexte, les contraintes, et exécute l'irréductible (publier, payer la pub, envoyer le DM).
- **Audience d'abord, données, distribution.** Le contenu n'est pas une oeuvre — c'est un actif : il doit servir l'offre et être mesurable.
- **Régularité > perfection au début.** Le volume construit la compétence **et** les données ; la perfection prématurée est une fuite.
- **Start ugly.** Les 30 premières pièces seront faibles — c'est le prix d'entrée. Objectif : apprendre vite, pas impressionner son ego.

## Process

### Phase 1 — Identifier le meilleur canal parmi les 6

Comparer honnêtement profil entrepreneur + offre + budget + audience cible. **Choisir UN canal.** Présenter la recommandation avec **raisons explicites** et ce qu'on ne fait pas (les 5 autres).

1. **Ads (pub payante)** — Facebook, Google, TikTok, etc.
   - **Pour :** budget disponible, besoin de résultats rapides, offre claire, validation minimale.
   - **Pas pour :** budget zéro, offre non validée, promesse floue.

2. **Content (création de contenu)** — YouTube, LinkedIn, TikTok, blog.
   - **Pour :** expertise à partager, à l'aise (ou prêt à s'entraîner) en création, horizon moyen.
   - **Pas pour :** besoin de cash immédiat, refus total de production, aucune capacité hebdo.

3. **SEO** — Blog, YouTube SEO.
   - **Pour :** patience 6-12 mois, contenu evergreen, souvent B2B / recherche intentionnelle.
   - **Pas pour :** urgence cash, marché hyper volatile, besoin de feedback rapide.

4. **Influenceurs** — Partenariats avec créateurs.
   - **Pour :** budget, produit visuel / consumer, peu de temps pour créer soi-même.
   - **Pas pour :** B2B complexe, budget zéro, tracking impossible.

5. **Affiliés** — Commissions sur ventes via partenaires.
   - **Pour :** marges suffisantes, produit digital ou reproductible, tracking fiable.
   - **Pas pour :** marges serrées, pas de système d'attribution, offre floue.

6. **Agence** — Déléguer l'acquisition.
   - **Pour :** budget solide, besoin de délégation, **déjà testé soi-même** (pour juger).
   - **Pas pour :** budget serré, jamais testé — sinon impossible de qualifier le travail.

**Décision :** scorer chaque option (fit audience, coût, délai, compétences). Sortie : **une** ligne du type : *« Canal retenu : [X] parce que [A, B, C]. On ne fait pas [Y, Z] pendant 90 jours. »*

### Phase 2 — Mettre en place le canal (BOS fait le gros du travail)

Appliquer les best practices du canal choisi. Ci-dessous : méthodologie complète si le canal = Content. Sinon, adapter.

#### Si le canal = Content — méthodologie complète

1. **Content-market fit**
   - **Audience :** qui exactement, où, quel langage, quelles peurs/objectifs.
   - **Pont vers l'offre :** quel CTA naturel (DM, call, lead magnet, essai).
   - **Capacité production :** heures/semaine, format toléré, contraintes (visage, voix, équipe).

2. **Choix de plateforme** — table de décision (à remplir dans le livrable) :

   | Critère | Plateforme A | Plateforme B |
   |--------|----------------|---------------|
   | Où est l'audience ? | | |
   | Format naturel pour l'entrepreneur | | |
   | Vitesse de feedback | | |
   | Concurrence / saturation | | |
   | Monétisation / B2B vs B2C | | |

   **Règle :** 1 plateforme principale pour les 90 jours (secondaire seulement si déjà maîtrisée).

3. **3-5 piliers de contenu** — intersection de : expertise réelle, problèmes audience, crédibilité pour l'offre. Chaque pilier : promesse en une phrase + 3 exemples de titres.

4. **Templates de format**
   - **LinkedIn :** Hook → Contexte → Insight → CTA.
   - **Shorts / Reels :** Hook → Setup → Payload → CTA.
   - **YouTube long :** Hook → Intro (promesse) → Contenu (segments) → CTA.

5. **Calendrier 7 jours** (pas 30 — itération hebdomadaire) : 1 ligne par jour (format, sujet, pilier, CTA).

6. **Workflow** : Idée → Hook → Production → Montage/editing → Publication → Engagement (réponses commentaires/DM) → mini-review hebdo (qu'est-ce qui a touché).

7. **Première pièce** : produite dans la session (brouillon prêt à poster ou script téléprompter).

#### Si le canal = Ads

- Structure de campagne (objectif, audiences, créatives, variantes).
- Ciblage (intention, démos, exclusions).
- Allocation budget (test vs scale, règles de couper/scale).
- Cadre de test créatif (hypothèse → variation → KPI → décision).

#### Si le canal = SEO

- Recherche mots-clés (intention, difficulté, quick wins).
- Plan éditorial aligné offre.
- Bases techniques (indexation, structure Hn, maillage interne — proportionné au niveau).

#### Autres canaux (influenceurs, affiliés, agence)

- **Influenceurs :** brief, critères de sélection, tracking, structure deal.
- **Affiliés :** grille commissions, conditions, assets, onboarding partenaires.
- **Agence :** scope, KPIs, reporting, garde-fous — seulement si prérequis budget + expérience perso respectés.

### Phase 3 — Première action maintenant

**Obligatoire :** un livrable #1 dans la session :
- copy / script / plan de pub / email / outline article / checklist SEO page 1, etc.

Si résistance (« pas prêt ») → réduire au plus petit incrément publiable, mais ne pas reporter à « plus tard » sans action tangible cette session.

### Phase 4 — Sauvegarder et suivre

1. **`Output/Traffic_Strategy_[date].md`** — synthèse : canal, raisons, setup, calendrier 7j si content, métriques à suivre, première pièce incluse.
2. **`Core/Actions.md`** — actions prioritaires (bold en tête), liées au problème trafic.
3. **`Core/Business.md`** — section Marketing : canal actif, promesse de mesure, contraintes 90 jours.
4. **`Core/Journal.md`** (si pertinent) — une ligne sur la décision canal + première action faite.

## Output

| Fichier | Contenu |
|--------|---------|
| `Output/Traffic_Strategy_[date].md` | Stratégie canal unique, setup, métriques, livrable #1 |
| `Core/Actions.md` | Actions court terme alignées trafic |
| `Core/Business.md` | Section Marketing à jour |

## Garde-fous

- **Ne JAMAIS proposer multi-canal aux débutants** — un canal, 90 jours, point.
- **Ne JAMAIS produire du contenu avant d'avoir défini l'audience et le pont vers l'offre** — sinon bruit et démotivation.
- **Ne JAMAIS envoyer l'entrepreneur « apprendre » tout seul** (cours, YouTube) — BOS enseigne en construisant dans la session.
- **Ne JAMAIS laisser « je suis pas prêt » gagner** — première pièce / pub / action dans cette session (ou micro-version irréfutable).
